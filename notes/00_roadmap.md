# Quant Learning Roadmap

## Background

- Mathematics major
- Some Python experience
- A little Stata
- Interested in quantitative investing and factor research

## Why I started this repo

I want a place to record my learning process in a structured way.  
Instead of learning scattered concepts, I hope to build a coherent path from basic factor ideas to simple backtesting projects.

## Stage 1

- restore Python workflow for data analysis
- practice pandas, returns, ranking, grouping
- get ready for factor experiments

## Stage 2

- understand what a factor is
- learn how to test a factor
- begin with momentum factor

## Stage 3

- compute IC and group returns
- build simple long-short logic
- understand Sharpe ratio, drawdown, turnover

## Stage 4

- write a small complete project
- improve results and interpretation
- turn experiments into a research-style summary
